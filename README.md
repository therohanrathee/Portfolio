# Rohan Rathee | Premium Developer Portfolio

Welcome to my personal developer portfolio, a high-performance, aesthetically pleasing web application designed to showcase my skills bridging the gap between Design and Engineering.

## 🌟 Key Features

- **Modern Glassmorphism Design**: Custom-built CSS modules featuring layered radial gradients, background blurs, and an immersive aesthetic.
- **Dynamic GitHub Integration**: Automatically fetches and displays my most recent, pinned GitHub repositories directly using the GitHub API, keeping projects always up-to-date.
- **Interactive Framer Motion Animations**:
  - Scroll-triggered entrance animations for a smooth storytelling experience.
  - Interactive "floating bubbles" physics for the Skills section.
  - Interactive micro-interactions across Bento Box layouts and CTA buttons.
- **Fully Responsive Architecture**: Carefully optimized layouts across all breakpoints, ensuring perfect framing from massive desktop displays to mobile screens.
- **Light & Dark Mode Ready**: Fully functional theme toggler ensuring a premium feel regardless of user preference.

## 💻 Tech Stack

- **Framework**: [Next.js 16.2.4 (App Router)](https://nextjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: Vanilla CSS (CSS Modules)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)

## 🚀 Getting Started

To run this project locally, follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/therohanrathee/portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open the application:**
   Navigate to [http://localhost:3000](http://localhost:3000) in your preferred browser.

## 📂 Project Structure

- `src/app/`: Next.js App Router definitions, global layouts, and server-side fetching.
- `src/components/`: Reusable React components (`Hero`, `About`, `Skills`, `Projects`, `Contact`) with their accompanying `.module.css` files.
- `public/`: Static assets such as SVGs, icons, and fonts.
